##### Raspi-GW / Wi-Fi AP 
- 라즈베리파이를 이용한 Wi-Fi AP 기능과 WSN (IoT) GW 동시 실행
  - Wi-Fi, USB dongle 을 사용함
  - raspberryPi 3 의 경우는 내장형 Wi-Fi 사용 
  - Kmote USB I/F 연결 
  - 안정성 확인, 시리얼 연동 확인
  - Wi-Fi 설정 제공
  - Raspi3 부터는 Wi-Fi, BLE 동시 지원
