### Detection Demo. 시스템
#### 동작절차
1. 카메라 촬영
2. 영상 전송
3. 영상 디텍션, 결과 연산
4. 결과 DB 저장
5. 결과 GUI 출력

#### 결과 화면
1. 측정 사진 영상
2. 연산결과 사진 (Box) 영상 출력
4. 해당 결과를 DB에 입력하고, 사용 숫자를 그래프로 출력

1. 데모용 Trigger 제작
RPI로 데모를 동작시키면 딜레이가 발생하기 때문에 BLE나 zigbee 버튼 모듈로 버튼을 누르면 데모 1루프 실행<br>
무언가 감지된 순간 사진을 찍는식 <br>

2. 여러장 촬영 후  2개의 RIP 영상분석
사진을 0.5초 단위로 2장 찍고 2장을 분석후 데이터 사진을 종합하여 DB에 저장<br>
결과를 GUI로 출력<br>
